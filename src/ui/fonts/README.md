# Bundled fonts

Self-hosted woff2 (no CDN at runtime — keeps the app offline/static). All three are
licensed under the **SIL Open Font License 1.1**, which permits bundling/redistribution:

- **Cinzel** (display / titles) — https://fonts.google.com/specimen/Cinzel
- **EB Garamond** (card + body text) — https://fonts.google.com/specimen/EB+Garamond
- **Oswald** (labels / banners) — https://fonts.google.com/specimen/Oswald

Files sourced from the `@fontsource` packages (latin subset). To refresh, re-download from
`https://cdn.jsdelivr.net/npm/@fontsource/<family>/files/<file>.woff2`.
