# Upgrade worklist — author pass

Every upgradeable card in the player's world, grouped by job. Write the **tier-1 text** on
each card's ➕ line. The rule: the upgrade is a **strictly stronger card in the same slot** —
a harder-hitting insult, a bigger brag, a shameless-er pander. It does NOT need to keep the
original's premise or wording; a brand-new joke that punches harder is better than the old
joke with more words. (Flagship cards can also get a ➕➕ tier-2 line — add one wherever a
second escalation writes itself.)

Grammar notes: attack/brag/pander predicate lines are shown conjugated for "my opponent" /
"I"-adjacent display, but author them so they'd work spoken by anyone (they can be self-owns).
Subject upgrades should stay usable mid-sentence exactly like the original.

Stat targets (applied when the texts land — you don't need to write numbers):
- Signature predicate ±3 → **±4**; the ±2s → **±3**
- Signature subject ×1.3 → **×1.6**; unloaded → **×1.3**
- Reward predicate/aside ±4 → **±5** (super-card tier)
- Reward subject ×1.6 → **×1.9** (super-card tier)

Not upgradeable (excluded): shared-pool commons and asides (never in your private deck),
connectors, finishers, power-ups.
